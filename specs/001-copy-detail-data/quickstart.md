# Quickstart: 詳細データのコピー

## 前提

- Bun（`tauri.conf.json` の `beforeDevCommand/beforeBuildCommand` が `bun run ...`）
- Tauri 2 / Rust toolchain

## セットアップ

1. 依存をインストール
   - `bun install`
2. 開発起動
   - `bun run tauri dev`

## テスト

- フロントエンド（Vitest）: `bun run test`
- Rust: `Push-Location src-tauri; cargo test; Pop-Location`

## ビルド

- `bun run tauri build`

---

## 手動検証手順

### US1: 詳細データのコピー

1. アプリを起動する（`bun run tauri dev`）
2. 集計タブを開く（プロジェクト別ビュー）
3. プロジェクトとブランチ、年を選択する
4. 詳細データパネル内のコピーアイコンをクリックする
5. スプレッドシートに貼り付ける
6. **確認事項**:
   - ヘッダー行 + 表示中の行がTSVで貼り付けられる
   - 列順が表示順と一致する
   - コピー完了メッセージがパネル上部に2秒表示される

### US2: 横断ビューでのコピー

1. 集計タブを開く（横断ビュー）
2. 年を選択する
3. 詳細データパネル内のコピーアイコンをクリックする
4. **確認事項**:
   - コピー内容が横断ビューの表示行と一致する

### US3: フィルタ/並び替え時のコピー

1. ユーザーフィルタで一部ユーザーのみ選択する
2. 詳細データパネルを確認する
3. コピーアイコンをクリックする
4. **確認事項**:
   - 表示されている行・順序のみがコピーされる

### US4: 0行時のコピー

1. 詳細データが0行になる条件を作る
2. コピーアイコンをクリックする
3. **確認事項**:
   - ヘッダー行のみがコピーされる
   - メッセージが表示される
